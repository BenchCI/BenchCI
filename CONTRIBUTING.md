# Contributing to BenchCI

Thank you for your interest in BenchCI.

BenchCI is a commercial product, and its core source code is maintained in a private repository.

---

## Contribution Model

At this time, external code contributions are not accepted.

However, contributions to the **public repository** are welcome in the following areas:

* documentation improvements
* example configurations
* bug reports and feedback
* feature suggestions

---

## Reporting Issues

If you encounter a problem while using BenchCI, please open an issue in this repository.

Include:

* description of the issue
* steps to reproduce
* board configuration (if applicable)
* test suite
* relevant logs (`transport.log`, `flash.log`, `gpio.log`)

---

## Suggesting Features

We welcome feedback and feature requests.

If you have ideas for improving BenchCI:

* open an issue
* describe your use case
* explain the expected behavior

---

## Documentation Contributions

You can help improve BenchCI by:

* fixing unclear sections
* improving examples
* suggesting better explanations

---

## Security Issues

For security-related concerns, do not open a public issue.

Follow the guidelines in SECURITY.md.

---
